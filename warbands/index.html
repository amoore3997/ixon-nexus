Warband Rules

Core construction limits

Warlord Required

Each Warband must include exactly one Warlord. The Warlord defines Command Points, faction identity, Coalition Keys, and Veto Flags.

Required

Influence Limit

The current prototype limit is 10 Influence. Standard units spend Influence from this pool, while the Warlord anchors the Warband.

10 Influence

Coalition Keys

Coalition Keys describe compatibility, identity, and shared battlefield logic. The current prototype roster uses Survival.

Survival 🛡

Veto Flags

Veto Flags prevent incompatible cards, factions, characters, or Coalition Keys from entering the same Warband.
